🧵 Loomwright

any task → the loop it needs

Loop engineering is the 2026 skill: the unit of agentic coding isn't the prompt, it's the loop — generate, check, revise, until a real exit condition holds. Everyone hand-builds those loops, and every coding-agent tool runs the same fixed pipeline for everything. Loomwright is a Band room that reads a task and synthesizes the loop that task needs — the checks that gate it, the critics it recruits, whether a human must sign — then runs it. Give it anything below and watch the loop change.
